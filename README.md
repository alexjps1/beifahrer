# Beifahrer

## Deutsch

Servus! Beifahrer ist ein LLM-basierter Agent, der während einer simulierten Autofahrt akkurate und hilfreiche Antworten zu Fahrassistenzsystemen liefert.

Dieses Repository enthält ein Backend, der API-Endpunkte für den Agenten bietet.
Diese können entweder von der React-Frontend (in diesem Repository) oder von dem Robotaxi-Konzept von Christian Schmidt abgerufen werden.

Entwickelt für den Lehrstuhl für Ergonomie der Technischen Universität München.

## English

Servus! Beifahrer is an LLM-based agent that provides accurate and helpful answers about driver assistance systems during a simulated drive.

This repository contains a backend that offers API endpoints for the agent. These can be accessed either from the React frontend (in this repository) or from Christian Schmidt's Robotaxi concept.

Developed for the Chair of Ergonomics at the Technical University of Munich.
